wiser — glasses lens mockups

Each frame is a true 600×600 lens — the device shows one view at a time, no scroll, a handful of words. Every dashboard view from ui-metrics.md is compressed to a single glanceable screen built from the same component hierarchy. Surfaces use #0a0a0f (never pure black — black is transparent on the lens). Test on desktop with arrow keys + Enter + Esc.

Component hierarchy (top → bottom, maps to MWDAT FlexBox/Text/Button): one root column, ≤6 elements, ≤2 actions.
